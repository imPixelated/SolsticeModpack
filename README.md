# SolsticeLauncher 🇧🇷
O SolsticeLauncher é um launcher de Minecraft desenvolvido como parte de um projeto pessoal, o SolsticeSMP. 

Seu principal objetivo é oferecer uma interface mais agradável, organizada e personalizada para acessar e gerenciar o jogo. Criado em Python, o launcher utiliza a biblioteca [Minecraft Launcher Lib](https://minecraft-launcher-lib.readthedocs.io/), que facilita a integração com os sistemas de autenticação, instalação e execução do Minecraft. A partir dessa base, o projeto busca não apenas cumprir sua função básica, mas também entregar uma experiência visual mais bonitinha e alinhada com a identidade do servidor.

Embora seja um projeto simples e sem pretensões comerciais, o SolsticeLauncher representa para nós uma exploração prática de desenvolvimento, unindo programação, design e personalização em um só lugar. Mais do que apenas um launcher, ele serve como uma extensão do próprio SolsticeSMP, reforçando a identidade do projeto e proporcionando aos usuários uma forma mais imersiva de iniciar sua jornada no servidor.

# SolsticeLauncher 🇺🇸
SolsticeLauncher is a custom Minecraft launcher developed as part of a personal project, SolsticeSMP. 
Its main goal is to provide a more visually appealing, organized, and personalized way to access and manage the game. Built with Python, the launcher leverages the [Minecraft Launcher Lib library](https://minecraft-launcher-lib.readthedocs.io/), which simplifies tasks such as authentication, installation, and game execution. On top of this foundation, the project focuses on delivering a smoother and more cohesive user experience aligned with the identity of the server.

While it is a relatively simple project with no commercial ambitions, SolsticeLauncher represents a practical exploration of development, combining programming, design, and customization in a single application. More than just a launcher, it acts as an extension of the SolsticeSMP experience, reinforcing the project's identity and offering users a more immersive way to start their journey.
